# Flarum2 Blog

`vadkuz/flarum2-blog` adds a blog section to your Flarum forum.

This fork is adapted for **Flarum 2.x beta.6**.

## Installation

```bash
composer require vadkuz/flarum2-blog
```

## Update

```bash
composer update vadkuz/flarum2-blog
```

## Notes

- Requires `flarum/core:^2.0.0-beta.6`
- Uses `flarum/tags` and `flarum/lock`
- Supports optional integrations with `fof/upload` and `fof/discussion-language`
